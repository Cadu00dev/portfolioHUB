# 🍄 Jogo: Mundo do Cogumelo

## 📖 Descrição
Este é um projeto de jogo interativo desenvolvido em Python. O objetivo principal foi aplicar na prática conceitos fundamentais de Engenharia de Software, especificamente a Programação Orientada a Objetos (POO), estruturação de código e lógica de programação. 

## 🚀 Tecnologias Utilizadas
* Python 3

## ⚙️ Funcionalidades
* Estruturação baseada em Classes e Objetos (POO).
* Controle de fluxo através de estruturas condicionais e laços de repetição.
* Interação via terminal com o jogador.

## 💻 Como Executar
1. Clone este repositório na sua máquina.
2. Navegue até o diretório `jogo-Mundo-do-cogumelo`.
3. Execute o arquivo principal via terminal:
   ```bash
   projeto.py
