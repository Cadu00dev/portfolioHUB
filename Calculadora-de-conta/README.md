
# 🧮 Calculadora de Conta

## 📖 Descrição
Uma aplicação de terminal desenvolvida para facilitar cálculos matemáticos do dia a dia, como a divisão de despesas ou a aplicação de taxas e descontos. Este projeto destaca a capacidade de criar ferramentas utilitárias eficientes através de código.

## 🚀 Tecnologias Utilizadas
* Python 3

## ⚙️ Funcionalidades
* Resolução de operações matemáticas básicas e avançadas.
* Formatação de dados de saída (exibição de valores monetários e *floats*).
* Interface simples via linha de comando (CLI).

## 💻 Como Executar
1. Clone este repositório na sua máquina.
2. Navegue até o diretório `Calculadora-de-conta`.
3. Execute o script principal via terminal:
   ```bash
   projeto2.py
