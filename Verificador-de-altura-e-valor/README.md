
# 📏 Verificador de Altura e Valor

## 📖 Descrição
Script automatizado focado em validação rigorosa de dados de entrada. O programa avalia condições específicas — como restrições de altura e checagem de valores financeiros — para validar ou recusar a solicitação de um usuário, garantindo que as regras de negócio sejam respeitadas.

## 🚀 Tecnologias Utilizadas
* Python 3

## ⚙️ Funcionalidades
* Validação de *inputs* (entradas do usuário).
* Utilização de lógica booleana e blocos `if/elif/else`.
* Tratamento de tipos de dados (conversão de *strings* para inteiros/floats).

## 💻 Como Executar
1. Clone este repositório na sua máquina.
2. Navegue até o diretório `Verificador-de-altura-e-valor`.
3. Execute o script principal via terminal:
   ```bash
    teste3.py
