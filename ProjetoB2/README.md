# 📅 Agenda de Tarefas (Projeto B2)

## 📖 Descrição
Este projeto é um gerenciador prático de tarefas via terminal (agenda de atividades). O programa permite que o usuário insira múltiplas atividades diárias de uma só vez, processando o texto e transformando-o automaticamente em uma lista organizada. O foco do código é a aplicação de lógica condicional e a manipulação de dados em Python.

## 🚀 Tecnologias Utilizadas
* Python 3

## ⚙️ Funcionalidades
* **Cadastro de atividades em lote:** O usuário pode adicionar várias tarefas simultaneamente separando-as por vírgula.
* **Manipulação de Strings e Listas:** Utilização do método `.split(",")` para fatiar o texto de entrada do usuário e estruturar os dados em uma lista.
* **Contagem de elementos:** Uso da função `len()` para calcular e exibir a quantidade total de atividades cadastradas.
* **Navegação condicional:** Controle do fluxo do programa (blocos `if/else`) para permitir que o usuário escolha entre cadastrar novas tarefas ou visualizar o status final da agenda.
